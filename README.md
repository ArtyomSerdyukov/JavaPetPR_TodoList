# 📝 Todo List Пэт-проект
Автор: Артём Сердюков
Простое веб-приложение для управления задачами (TODO List), построенное на Spring Boot.

## 🛠️ Технологии (Tech Stack)

### Backend
- **Spring Boot 3.x** - фреймворк для создания приложений
- **Spring Data JPA** - работа с базами данных
- **Hibernate** - ORM для работы с объектами БД
- **PostgreSQL** - система управления базами данных

### Frontend
- **Thymeleaf** - шаблонизатор для генерации HTML
- **Bootstrap 5** - CSS-фреймворк для стилизации

### Инструменты разработки
- **IntelliJ IDEA** - IDE для разработки
- **Maven** - управление зависимостями
- **Lombok** - сокращение boilerplate кода
- **Git** - система контроля версий


Функциональность

✅ Создание новых задач

✅ Просмотр списка задач

✅ Отметка задач как выполненных

✅ Удаление задач
